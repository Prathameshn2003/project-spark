## 🛠️ How can I edit this code?

There are several ways to work on your NaariCare application:

---

### 💻 Use Your Preferred IDE (Recommended)

If you want full control and real development experience:

#### Steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Open project folder
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start development server
npm run dev
```

---

### ✏️ Edit Directly in GitHub

* Open your repository
* Select the file you want to edit
* Click the ✏️ Edit button
* Commit changes

---

### ☁️ Use GitHub Codespaces

* Go to repository main page
* Click **Code → Codespaces**
* Click **New Codespace**
* Start editing instantly in browser
